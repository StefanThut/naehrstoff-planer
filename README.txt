Nährstoff-Planer – Modular (Starter-Kit)
======================================
So verwendest du das Paket
1) Lade die ZIP herunter und entpacke sie.
2) Öffne index.html im Browser (Desktop/Handy). Kein Build nötig.
3) Daten bleiben lokal (localStorage).

Dateien
- index.html, style.css
- utils.js, storage.js, data.js
- ui-tabs.js, users.js, foods.js, menu.js, alternatives.js, distribution.js, main.js

Anpassen
- Styles in style.css, Logik in den jeweiligen *.js-Dateien.
- Mehr Lebensmittel: data.js → foods-Array.
- Sync/PWA kannst du später ergänzen (optionale Erweiterung).
